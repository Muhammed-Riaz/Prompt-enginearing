# Code Review Prompt Design - GIAIC Assignment

## Prompt 1: Basic Code Review
You are a senior software engineer. Review the following code and identify bugs, bad practices and code smells. Also suggest improvements in simple points.

## Prompt 2: Performance Optimization
Analyze this code and suggest ways to improve its performance and reduce memory usage. Also explain why your suggestions are better.

## Prompt 3: Security Check
Review this code from a security point of view. Identify any vulnerabilities and give secure alternatives.

## Prompt 4: Code Refactoring
Refactor the following code using best practices and clean code principles. Keep it simple and readable for a beginner developer.

## Prompt 5: Documentation & Comments
Add proper comments and documentation to this code so that a beginner can easily understand it.
